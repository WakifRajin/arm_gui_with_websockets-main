## CONTROLS

LT and RT :  shared motor pwm slider
LB and RB : wrist servo

D pad up : gripper open
D pad down : gripper closed
D pad right : roller open
D pad left : roller closed
L button : gripper stop
R button : roller stop

B : base motor toggle
X : shoulder toggle
Y : elbow toggle

START : reset
